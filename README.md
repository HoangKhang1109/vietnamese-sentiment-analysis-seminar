# vietnamese-sentiment-analysis-seminar
# Phân Loại Cảm Xúc Tiếng Việt – PhoBERT

Ứng dụng web phân loại cảm xúc tiếng Việt (Tích cực / Tiêu cực / Trung tính) sử dụng mô hình **PhoBERT** fine-tuned.

### Demo trực tiếp
https://vietnamese-sentiment-sgu-seminar.streamlit.app

### Tính năng
- Phân tích tức thì, độ chính xác cao
- Lưu lịch sử phân tích (SQLite)
- Nút xóa lịch sử có xác nhận
- Giao diện đẹp, responsive

### Công nghệ
- PhoBERT (vinai/phobert-base)
- Streamlit
- PyTorch + Transformers
